**Задание 00**

Напиши SQL-запрос, который возвращает:

- название пиццы;
- цену;
- название пиццерии;
- дату посещения.

Выведи данные для клиента **Kate** с ценами в диапазоне от **800** до **1000** рублей.

Отсортируй результат по названию пиццы, цене и названию пиццерии.

---

**Задание 01**

Найди все идентификаторы меню, которые никто не заказывал.

Отсортируй результат по идентификатору.

**Запрещено:** любые типы `JOIN`.

---

**Задание 02**

Используй SQL-запрос из **Задания 01** и выведи названия пицц, которые никто не заказывал, включая соответствующие цены и названия пиццерий.

Отсортируй результат по названию пиццы и цене.

---

**Задание 03**

Найди пиццерии, которые женщины посещали чаще мужчин или мужчины посещали чаще женщин.

При использовании операторов работы с множествами (`UNION ALL`, `EXCEPT ALL`, `INTERSECT ALL`) сохраняй дубликаты.

Отсортируй результат по названию пиццерии.

---

**Задание 04**

Найди объединение пиццерий:

- в которых заказы делали только женщины;
- в которых заказы делали только мужчины.

Используй операцию `UNION` над этими множествами.

При использовании операторов множеств (`UNION`, `EXCEPT`, `INTERSECT`) не включай дубликаты.

Отсортируй результат по названию пиццерии.


**Задание 00**

Напиши SQL-запрос, который объединит в один общий список идентификаторы меню и названия пицц из таблицы `menu`, а также идентификаторы и имена людей из таблицы `person`.

В итоговом результате столбцы должны называться `object_id` и `object_name`. Сначала должна идти выборка по `object_id`, затем по `object_name`.

---

**Задание 01**

Измени SQL-запрос из **Задания 00** следующим образом:

- Удали столбец `object_id`.
- Измени порядок сортировки: сначала выведи данные из таблицы `person` (по `object_name`), затем из таблицы `menu` (также по `object_name`).
- Сохрани дублирующиеся строки.

---

**Задание 02**

Напиши SQL-запрос, который:

- Возвращает уникальные названия пицц из таблицы `menu`.
- Сортирует результаты по столбцу `pizza_name` в порядке убывания.

**Запрещено:** `DISTINCT`, `GROUP BY`, `HAVING`, все виды `JOIN`.

---

**Задание 03**

Напиши SQL-запрос, который:

- Находит пересечение данных по дате заказа (`order_date`) и `person_id` из таблицы `person_order`.
- Находит пересечение данных по дате визита (`visit_date`) и `person_id` из таблицы `person_visits`.

Иными словами, необходимо найти идентификаторы людей, которые в один и тот же день и посетили пиццерию, и сделали заказ.

Отсортируй результат сначала по `action_date` в порядке возрастания, затем по `person_id` в порядке убывания.

**Запрещено:** все виды `JOIN`.

---

**Задание 04**

Напиши SQL-запрос, который вернет разницу (операция `MINUS`) по значениям столбца `person_id` между таблицами `person_order` и `person_visits`, сохраняя дубликаты.

Учитывай только записи за **7 января 2022 года**:

- `order_date = '2022-01-07'`
- `visit_date = '2022-01-07'`

**Запрещено:** все виды `JOIN`.

---

**Задание 05**

Напиши SQL-запрос, который вернет все возможные комбинации записей из таблиц `person` и `pizzeria`.

В результате сначала должны идти столбцы из таблицы `person`, затем — из таблицы `pizzeria`.

**Задание 06**

Измени SQL-запрос из **Задания 03** так, чтобы он возвращал имена людей вместо их идентификаторов.

Измени сортировку:

- сначала по `action_date` в порядке возрастания;
- затем по `person_name` в порядке убывания.

**Задание 07**

Напиши SQL-запрос, который возвращает:

- дату заказа из таблицы `person_order`;
- имя человека, сделавшего заказ (из таблицы `person`), при этом имя и возраст должны быть отформатированы как `Andrey (age:21)`.

Отсортируй результат по обоим столбцам в порядке возрастания.

---

**Задание 08**

Перепиши SQL-запрос из **Задания 07**, используя конструкцию `NATURAL JOIN`.

Результат должен полностью совпадать с результатом из **Задания 07**.

**Запрещено:** все виды `JOIN` (кроме `NATURAL JOIN`).

**Задание 09**

- первый — с использованием оператора `IN`;

**Задание 10**

Напиши SQL-запрос, который вернет список людей, заказавших пиццу в соответствующей пиццерии.

Результат должен содержать:

- имя клиента;
- название пиццы;
- название пиццерии.

Отсортируй результат по трем столбцам в порядке возрастания:

- `person_name`;
- `pizza_name`;
- `pizzeria_name`.

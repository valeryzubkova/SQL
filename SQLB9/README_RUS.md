**Задание 00**

Напиши SQL-запрос, который возвращает:

- `person_id`;
- количество посещений любых пиццерий (`count_of_visits`).

Отсортируй результат:

- по количеству посещений — по убыванию;
- затем по `person_id` — по возрастанию.

---

**Задание 01**

Измени SQL-запрос из **Задания 00** так, чтобы вместо `person_id` возвращалось имя человека (`name`).

Выведи только **топ-4** человека с наибольшим количеством посещений.

Отсортируй результат по имени человека.

---

**Задание 02**

Напиши SQL-запрос, который выводит **топ-3** пиццерии:

- по количеству посещений;
- по количеству заказов.

Объедини результаты в один список.

Добавь столбец `action_type`, содержащий:

- `order` — если данные получены из заказов;
- `visit` — если данные получены из посещений.

Отсортируй результат:

- по `action_type` — по возрастанию;
- по количеству (`count`) — по убыванию.

---

**Задание 03**

Используя запрос из **Задания 02**, вычисли для каждой пиццерии:

- общее количество заказов;
- общее количество посещений;
- суммарное количество (`total_count`).

Учитывай, что пиццерия может присутствовать только в одной из таблиц.

Отсортируй результат:

- по `total_count` — по убыванию;
- затем по названию пиццерии (`name`) — по возрастанию.

**Задание 04**

- имя человека;

Выведи только тех людей, которые посещали пиццерии **более трех раз**.

**Задание 05**

**Задание 06**

**Задание 08**

- адрес человека;
- название пиццерии;
- количество заказов (`count_of_orders`) в этой пиццерии.

- по адресу;
- затем по названию пиццерии.

**Задание 09**

Напиши SQL-запрос, который возвращает агрегированную информацию по каждому адресу.

Результат должен содержать:

- адрес;
- вычисляемое значение по формуле:

  `Максимальный возраст - (Минимальный возраст / Максимальный возраст)`;

- средний возраст (`average`);
- результат сравнения формулы со средним возрастом (`comparison`):

  - `true`, если значение формулы больше среднего возраста;
  - `false` — иначе.

Отсортируй результат по адресу.

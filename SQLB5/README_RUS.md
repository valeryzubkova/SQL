**Задание 04**

Напиши SQL-запрос, реализующий формулу:

`(R − S) ∪ (S − R)`

где:

- `R` — таблица `person_visits` с фильтром по дате **2 января 2022 года**;
- `S` — таблица `person_visits` с фильтром по дате **6 января 2022 года**.

Вычисления должны выполняться над множеством значений столбца `person_id`.

Результат должен содержать только столбец `person_id`.

Отсортируй результат по `person_id` и сохрани итоговый запрос в представлении `v_symmetric_union`.

**Задание 05**

Создай представление `v_price_with_discount`, которое возвращает:

- имя клиента (`name`);
- название пиццы (`pizza_name`);
- реальную цену (`price`);
- столбец `discount_price`, рассчитанный по формуле:

  `price - price * 0.1`

Приведи столбец `discount_price` к типу `integer`.

Отсортируй результат по:

- `name`;
- `pizza_name`.


**Задание 00**

Напиши SQL-запрос, который выведет список пиццерий с их рейтингом, но только тех, которые никто не посещал.

**Запрещено:** `NOT IN`, `IN`, `NOT EXISTS`, `EXISTS`, `UNION`, `EXCEPT`, `INTERSECT`.

---

**Задание 01**

Напиши SQL-запрос, который возвращает отсутствующие даты с **1 по 10 января 2022 года** (включительно) для посещений людьми с идентификаторами **1** и **2** — то есть дни, пропущенные обоими.

Отсортируй результат по дате посещения в порядке возрастания.

**Запрещено:** `NOT IN`, `IN`, `NOT EXISTS`, `EXISTS`, `UNION`, `EXCEPT`, `INTERSECT`.

---

**Задание 02**

Напиши SQL-запрос, который возвращает:

- полный список имен людей, посетивших (или не посетивших) пиццерии в период с **1 по 3 января 2022 года**;
- полный список названий пиццерий, которые посещались (или не посещались) в этот же период.

Для `NULL`-значений в столбцах `person_name` и `pizzeria_name` используй замену на `'-'`.

Добавь сортировку по всем трем столбцам.

**Запрещено:** `NOT IN`, `IN`, `NOT EXISTS`, `EXISTS`, `UNION`, `EXCEPT`, `INTERSECT`.

---

**Задание 03**

Перепиши SQL-запрос из **Задания 01**, используя `CTE` (`Common Table Expression`).

В `CTE` реализуй генератор дат.

Результат должен совпадать с результатом из **Задания 01**.

**Запрещено:** `NOT IN`, `IN`, `NOT EXISTS`, `EXISTS`, `UNION`, `EXCEPT`, `INTERSECT`.
